Creates and outputs a new \mbox{\hyperlink{class_g_w_1_1_g_r_a_p_h_i_c_s_1_1_g_direct_x11_surface}{GDirect\+X11\+Surface}} object. 

Initializes a handle to a \mbox{\hyperlink{class_g_w_1_1_g_r_a_p_h_i_c_s_1_1_g_direct_x11_surface}{GDirect\+X11\+Surface}} object with an existing GWindow. The created \mbox{\hyperlink{class_g_w_1_1_g_r_a_p_h_i_c_s_1_1_g_direct_x11_surface}{GDirect\+X11\+Surface}} object will have its reference count initialized to one and register as a listener to the provided GWindow object.

This function accepts a bit mask that can hold supported \textquotesingle{}GGraphics\+Init\+Options\textquotesingle{}, which will be taken into account when creating the context. To ignore this mask, simply pass in 0 when calling this function and the context will be created with default settings.
