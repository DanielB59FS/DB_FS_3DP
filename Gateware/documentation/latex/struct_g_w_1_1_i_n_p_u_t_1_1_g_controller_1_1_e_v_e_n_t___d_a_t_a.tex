\doxysubsection{Detailed Description}
\mbox{\hyperlink{struct_g_w_1_1_i_n_p_u_t_1_1_g_controller_1_1_e_v_e_n_t___d_a_t_a}{EVENT\+\_\+\+DATA}} holds the information about a controller event. See \mbox{\hyperlink{group___g_controller_codes}{GController\+Codes}} for a list of input\+Code defines.

If the corresponding event is an Input Event input\+Code and input\+Value will contain the input code and value respectively for the detected input. If the corresponding event is an Connection event input\+Code and input\+Value will be 0. is\+Connected will be 0 for false and 1 for true.
