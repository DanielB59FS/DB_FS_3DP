98 /*!
99 * Initializes a handle to a GDirectX12 object with an existing GWindow.
100 * The created GDirectX12 object will have its reference count initialized
101 * to one and register as a listener to the provided GWindow object.
102 *
103 * This function accepts a bit mask that can hold
104 * supported 'GGraphicsInitOptions', which will
105 * be taken into account when creating the context.
106 * To ignore this mask, simply pass in 0 when calling
107 * this function and the context will be created with
108 * default settings.
109 *
110 * NOTE: Supported initMask's are: GW::GRAPHICS::COLOR_10_BIT | GW::GRAPHICS::DEPTH_BUFFER_SUPPORT | GW::GRAPHICS::DEPTH_STENCIL_SUPPORT
111 *
112 * \param [in] _gwindow A object to an existing valid GWindow object.
113 * \param [in] _initMask The bit mask that can hold special initialization options.
114 *
115 * \retval GReturn::INVALID_ARGUMENT: _gwindow is invalid.
116 * \retval GReturn::FAILURE: A GDirectX12 object was not created.
117 * \retval GReturn::SUCCESS: A GDirectX12 object was successfully created.
118 */
