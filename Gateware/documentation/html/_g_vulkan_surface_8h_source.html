162 //! Creates and outputs a new GVulkanSurface object, along with giving more control to add extensions and layers
163 /*!
164 * Initializes a handle to a GVulkanSurface object with an existing GWindow.
165 * The created GVulkanSurface object will have its reference count initialized
166 * to one and register as a listener to the provided GWindow object.
167 *
168 * This function accepts a bit mask that can hold
169 * supported GGraphicsInitOptions, which will
170 * be taken into account when creating the context.
171 * To ignore this mask, simply pass in 0 when calling
172 * this function and the context will be created with
173 * default settings.
174 *
175 * This function accepts 6 additional parameters. This allows the ability to
176 * add additional functionality to your vulkan surface. This include additional
177 * debugging layers and additional extensions based on what your GPU can do.
178 *
179 *
180 * \param [in] _gwindow A pointer to an existing GWindow object.
181 * \param [in] _initMask The bit mask that can hold special initialization options.
182 * \param [in] _layerCount The count on how many layers you are providing to the vulkan instance.
183 * \param [in] _layers The list of Validation Layers to enable within the vulkan instance
184 * \param [in] _instanceExtensionCount The count on how many instance extensions you are providing to the vulkan instance
185 * \param [in] _instanceExtensions The list of Instance Extensions to enable within the vulkan instance
186 * \param [in] _deviceExtensionCount The count on how many device extensions you are providing to the vulkan logical device
187 * \param [in] _deviceExtensions The list of device extensions to enable within the vulkan logical device
188 * \param [in] _allPhysicalDeviceFeatures Bool if you want to enable all supported features of your GPU
189 *
190 * \retval GReturn::INVALID_ARGUMENT _gwindow is invalid, or there is a bad match of "count" and layers/extension provided (ex: 1 count but no layer/extension, or 0 count yet a layer/extension was sent.)
191 * \retval GReturn::FAILURE A GVulkanSurface object was not created.
192 * \retval GReturn::SUCCESS A GVulkanSurface object was successfully created.
193 */
194 GReturn Create(GW::SYSTEM::GWindow _gwindow, unsigned long long _initMask, unsigned int _layerCount, const char** _layers, unsigned int _instanceExtensionCount, const char** _instanceExtensions, unsigned int _deviceExtensionCount, const char** _deviceExtensions, bool _allPhysicalDeviceFeatures);

199 RELEASE_RESOURCES //!< Internal Vulkan resources are about to be destroyed.Destroy any child objects.
