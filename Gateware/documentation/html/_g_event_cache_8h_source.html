49 //! A Proxy class that can be used to record and playback GEvents captured from GEventGenerators.
50 /*!
51 * Stores all events produced by an interface derived from GEventGenerator that you are Registered to.
52 * **NOTE: GEventCache is a replacement proxy to GEventQueue which is deprecated.**
53 * \code{.cpp}
54 * GEventCache myCache;
55 * myCache.Create( 32 ); // can hold up to 32 unique GEvents, try to keep this as small as needed
56 * someEventGenerator.Register(myCache);
57 * // ... some recurring function or thread where you wish to process waiting events ...
58 * GEvent g;
59 * while (+myCache.Pop(g)) // if you wait too long to process events the oldest ones may be lost.
60 * {
61 * GEventGenerator::Events e; // substitute the generator type you care about
62 * GEventGenerator::EVENT_DATA d; // substitute the event data you care about
63 * // Determine if the "Pop"ed event contains the data you are intrested in.
64 * if (+g.Read(e,d)) // if you are interested in multiple events consider using a switch below
65 * if (e == GEventGenerator::Events::NON_EVENT) // substitute actual event
66 * std::cout << d.noData; // substitute actual data
67 * }
68 * \endcode
69 */

89 //! Allocates & Intializes a GEventCache.
90 /*!
91 * A GEventCache is capable of receiving and storing events created by any GEventGenerator.
92 * You can use this class whenever there is a Gateware class that spawns events you wish record.
93 * Unlike GEventResponder the GEventCache is designed to save multiple events up to a limit defined by you.
94 * If you need to process events on a specific thread, you can use this class to delay processing until
95 * a safe time to do so. Just make sure there is adequate capcity in the Cache, and "Pop" them off when ready.
96 *
97 * \param [in] _cacheSize The maximum amount of GEvents that can be retained by this instance. (must be > 0)
98 * Exceeding this amount without processing a message will cause the oldest to missed.
99 *
100 * \retval GReturn::INVALID_ARGUMENT A valid Cache Capacity > 0 must be provided.
101 * \retval GReturn::FAILURE Allocation has failed, you don't have enough memory.
102 * \retval GReturn::SUCCESS The cache was succesfully allocated, you can now Register it to GEventGenerators.
103 */
104 GReturn Create(unsigned int _cacheSize);
