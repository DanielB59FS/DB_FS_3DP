enum class GW::MATH2D::GCollision2D::GCollisionCheck2D
strong

Listing of common codes returned by select GCollision2D functions.

Values that represent collision checks

Enumerator
ERROR_NO_RESULT 

Used internally by Gateware for testing purposes

ABOVE 

An object is above another object

BELOW 

An object is below another object

NO_COLLISION 

There is no collision between two objects

COLLISION 

There is a collision between two objects
