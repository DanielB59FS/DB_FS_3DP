129 /*! EVENT_DATA holds the information about a controller event. See \ref GControllerCodes for a list of inputCode defines.
130 *
131 * If the corresponding event is an Input Event inputCode and inputValue will contain the input code and value
132 * respectively for the detected input. If the corresponding event is an Connection event inputCode and inputValue will be 0.
133 * isConnected will be 0 for false and 1 for true.
134 */
135 struct EVENT_DATA

144 //! Used to poll the current state of a button or axis on a controller
145 /*!
146 * Use an Inputcode to check the state of a button or axis on the controller
147 * at the selected index. If a button is being checked the _outState will be 0 for
148 * up and 1 for down. If a axis is being checked the _outState will be between -1 and 1 inclusively.
149 *
150 * \param [in] _controllerIndex The controller index to check
151 * \param [in] _inputCode The input code for the button/axis to check
152 * \param [out] _outState A reference to a float to store the state.
153 *
154 * \retval GReturn::INVALID_ARGUMENT Ether _controllerIndex is out of range or _inputCode is invaild.
155 * \retval GReturn::FAILURE No controller is connected at the chosen index.
156 * \retval GReturn::EMPTY_PROXY The proxy is empty
157 * \retval GReturn::SUCCESS The button/axis state was was successfully stored in the out-param
158 */

218 *
219 * \retval GReturn::FEATURE_UNSUPPORTED Vibration for the current controller type is not supported.
220 * \retval GReturn::INVALID_ARGUMENT One or more arguments are out of range.
221 * \retval GReturn::FAILURE Controller is currently vibrating.
