enum class GW::GReturn : int
strong

Listing of common failure/success codes returned by Gateware functions.

Enumerator
END_OF_FILE 

The requested file has no more data to read in.

FORMAT_UNSUPPORTED 

The requested file format is not supported by the current platform/object.

NO_IMPLEMENTATION 

An interface function was called directly without a valid implementation.

HARDWARE_UNAVAILABLE 

This feature requires hardware/drivers that are currently not present.

DEADLOCK 

The thread on which you called this function would cause a deadlock.

UNEXPECTED_RESULT 

A function has reached a spot it was never expected to be able to. (Report these!)

RESOURCE_LOCKED 

This operation has failed because a shared resource could not be aquired.

DISCONNECTED 

This operation has failed because you are no longer connected.

PREMATURE_DEALLOCATION 

You have deallocated some Interface this object was using internally. (Hint::Share())

FUNCTION_DEPRECATED 

This function has been disabled but has not been removed.

FEATURE_UNSUPPORTED 

Attempted an operation that is currently not supported.

FILE_NOT_FOUND 

A requested file is not found.

INTERFACE_UNSUPPORTED 

The requested interface is not supported by the current platform/object.

MEMORY_CORRUPTION 

A memory corruption occurred in the function.

INVALID_ARGUMENT 

An invalid argument was passed into the function.

IGNORED 

The requested operation has already been performed and cannot be done again.

EXPIRED_PROXY 

The Interface referenced by this Proxy has been deallocated. (C++17 future support)

FAILURE 

The function failed (Check function documentation for possible reasons).

EMPTY_PROXY 

This Proxy was never (or is no longer) initialized to a valid Interface.

SUCCESS 

The function succeeded.

REDUNDANT 

The function succeeded but was not necessary.
