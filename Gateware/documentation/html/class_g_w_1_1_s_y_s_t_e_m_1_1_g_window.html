◆ Events

enum class GW::SYSTEM::GWindow::Events
strong

Events holds the possible events a GWindow window can broadcast.

Enumerator
MINIMIZE 

Window was minimized.

MAXIMIZE 

Window was maximized.

RESIZE 

Window was resized in some way.

MOVE 

Window was moved.

DISPLAY_CLOSED 

Linux only message that is fired after XCloseDisplay() but before XCloseWindow()

EVENTS_PROCESSED 

Notifies users that all queued window events have been processed.

DESTROY 

Window was closed.

◆ ProcessWindowEvents()

virtual GReturn GW::SYSTEM::GWindow::ProcessWindowEvents ( )
pure virtual

Flushes and processes all messages from the window's event queue.

This function is meant to be called once a frame in an application's main loop. This function will break when all waiting messages have been processed and the event queue is empty.

Return values
GReturn::FAILUREThe window is invalid or is about to be invalid
GReturn::SUCCESSThe messages were successfully processed and removed

virtual GReturn GW::SYSTEM::GWindow::ReconfigureWindow ( int  _x,

int  _width,

GW::SYSTEM::GWindowStyle  _style 
)
pure virtual

Gives the currently opened window the specified size, position and style.

If width and height are equal to or greater than the native resolution, the passed in GWindowStyle will be overwritten to be the fullscreen version if it is not already. If position parameters are less than 0 then 0 will be used. If position parameters are greater than native resoultion, maximum native resolution parameters will be used.

Parameters
[in]_xThe x position on screen to place the top left corner of the window.
[in]_yThe y position on screen to place the top left corner of the window.
[in]_widthThe width to resize the client area of the window to.
[in]_heightThe height to resize the client area of the window to.
[in]_styleThe style to give to the window. (see GWindowStyle for style options)
Return values
GReturn::FAILUREThe window is invalid
GReturn::SUCCESSThe operation is successful
